# Website-Pessoal
Bem-vindo ao meu repositório do meu portfólio pessoal! Este é um espaço dedicado a documentar minha trajetória acadêmica e meus projetos técnicos na área de tecnologia.
Observação: O meu website está sujeito a evolução e manutenções futuras!

# Sobre o projeto
Este site foi desenvolvido para ser minha "vitrine" digital. Ele apresenta quem eu sou, minhas áreas de interesse (Web, Software e Automação) e links diretos para minhas redes profissionais.

# Tecnologias utilizadas:
* HTML5
* CSS3
* Flexbox: Organização responsiva do container de perfil
* Google Fonts
